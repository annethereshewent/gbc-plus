How to use

To load a game, simply hit "Load game" and choose a game!

Keyboard hotkeys:

  • Change DMG color palette
  • Show waveform visualizer
  • Quick save state
  • Quick load state

PS5 hotkeys:

  • Quick save state
  • Quick load state

Controller mappings can be changed from the "misc" menu.
The joypad mappings are based on the PS5 DualSense controller.
